### 一、本地回滚

```bash
git reset --hard commit-id   # 回滚到commit-id
```

### 二、远程回滚

1. `git checkout main`
2. `git pull`
3. `git branch main_backup`   # 备份分支
4. `git reset --hard the_commit_id`   # 本地回滚
5. `git push origin :master`   # 删除远程master
6. `git push origin master`   # 重新推送
7. `git branch -d master_backup`   # 删除备份
